# 1. Inicia um repositório Git local git init # 2. Adiciona todos os seus arquivos para serem monitorados git add . # 3. Cria o primeiro "commit" (um snapshot do seu projeto) git commit -m "Versão inicial do site" # 4. Renomeia a branch principal para "main" (padrão atual) git branch -M main # 5. Conecta seu repositório local com o repositório remoto no GitHub # (Substitua a URL pela URL do seu repositório criado no passo anterior) git remote add origin https://github.com/SEU-USUARIO/site-drgabriel.git # 6. Envia seus arquivos para o GitHub git push -u origin main
